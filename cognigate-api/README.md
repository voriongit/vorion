# Cognigate Engine API

The Cognigate Engine is VORION's governance runtime for AI agents. It implements the BASIS standard through three core layers:

- **INTENT** - Goal processing and risk assessment
- **ENFORCE** - Policy validation and gating
- **PROOF** - Immutable audit ledger

## Quick Start

```bash
pip install -e .
uvicorn app.main:app --reload
```

API docs: http://localhost:8000/docs
