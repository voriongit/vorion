## 1. BASIS Trust Framework

### Trust Tiers (8 levels, 0-1000 score range)

| Tier | Name | Score Range | Description |
|------|------|-------------|-------------|
| T0 | Sandbox | 0-199 | Initial testing, no real-world access |
| T1 | Observed | 200-349 | Monitored execution, limited capabilities |
| T2 | Provisional | 350-499 | Conditional trust, expanding capabilities |
| T3 | Verified | 500-649 | Verified track record |
| T4 | Operational | 650-799 | Production-ready, autonomous within scope |
| T5 | Trusted | 800-875 | High trust, minimal oversight |
| T6 | Certified | 876-949 | Certified for sensitive operations |
| T7 | Autonomous | 950-1000 | Full autonomy within governance bounds |

## 3. Cognigate TypeScript SDK

### Published Package
- NPM: [@vorionsys/cognigate](https://www.npmjs.com/package/@vorionsys/cognigate)
- Version: 1.0.0

### Installation
```bash
npm install @vorionsys/cognigate
```

### Usage
```typescript
import { Cognigate } from '@vorionsys/cognigate';

const client = new Cognigate({
  apiKey: process.env.COGNIGATE_API_KEY,
});

// Get trust status
const status = await client.trust.getStatus('agent-123');

// Evaluate governance
const { intent, result } = await client.governance.evaluate(
  'agent-123',
  'Read customer data'
);
```

### SDK Features
- Full TypeScript support
- Zod runtime validation
- Trust management (getStatus, getHistory, submitOutcome)
- Governance (parseIntent, enforce, evaluate, canPerform)
- Proofs (get, list, getStats, verify)
- Agents (list, get, create, update, delete, pause, resume)
- Webhooks (WebhookRouter, signature verification)

### Files
- `packages/cognigate/src/client.ts` - Main client
- `packages/cognigate/src/types.ts` - Type definitions
- `packages/cognigate/src/webhooks.ts` - Webhook utilities
- `packages/cognigate/src/index.ts` - Exports
- `packages/cognigate/README.md` - Full documentation
